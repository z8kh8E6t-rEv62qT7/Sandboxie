Классический пользовательский интерфейс Sandboxie (SbieCtrl.exe) имеет

установить новый UI Sandboxie-Plus (SandMan.exe),
который обеспечивает отличные возможности отслеживания и мониторинга.

6002;txt;01

.

6003;txt;01
Устаревший пользовательский интерфейс Sandboxie classic не поддерживает функции сетевого брандмауэра.

Однако вы можете настроить возможность, поддерживаемую основными компонентами, с помощью файла ini, правила имеют следующую структуру:
NetworkAccess=iexplorer.exe,Allow; Port=80,443; Address=192.168.0.1-192.168.100.255; Protocol=TCP

Если вы предпочитаете использовать UI для управления этими параметрами, перейдите на Sandboxie-Plus, который имеет полную поддержку UI для всех новых функций.
.

6004;pop;err;01

.

6005;txt;01
Примечание: Без прав администратора установщики не запустятся в песочнице. Sandboxie-Plus предлагает возможность добиться успеха многим установщикам без этих привилегий. Вы можете включить эту опцию и в классической сборке вручную, добавив "FakeAdminRights=y" в раздел ini для этой песочницы.
.

6006;txt;01
Устаревший пользовательский интерфейс Sandboxie classic поддерживает только несколько параметров ограничений.

Дополнительные параметры, такие как "ClosePrintSpooler=y", "OpenClipboard=n", "BlockNetParam=n" и многие другие, можно установить в разделе ini для этой песочницы.
Современный UI (SandMan.exe) также предлагает дополнительные параметры настройки для параметров "Запуск/Выполнение" и ограничения доступа в Интернет.

Если вы предпочитаете использовать UI для управления этими параметрами, перейдите на Sandboxie-Plus, который имеет полную поддержку пользовательского интерфейса для всех новых функций.
.

6007;txt;01
Sandboxie Plus предлагает улучшенную защиту конфиденциальности, переключая старое поведение с режима черного списка, т.е. разрешая доступ для чтения ко всему диску, за исключением случаев, заблокированных закрытым путем или путем записи, в режим белого списка, где изолированным программам разрешено только чтение общих системных местоположений и доступ для чтения к большинству пользовательских данных должен быть сначала явно предоставлен.

Устаревший UI Sandboxie classic не поддерживает этот режим работы. Хотя его можно настроить с помощью файла ini, рекомендуется использовать современный пользовательский интерфейс Sandboxie-Plus (SandMan.exe), если вы хотите использовать песочныцы с повышенной конфиденциальностью.
